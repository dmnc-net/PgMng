pgmng
=====

Pigmap Manager

INSTALLATION
1) # chmod +x pgmng_install
2) # ./pgmng_install

USAGE
1) open pgmng.cfg and follow your instincts
2) run pgmng in the desired mode:
    # ./pgmng full
    # ./pgmng incr
